# Full Example

This example demonstrates every KvikkPress feature. Use it as a reference when building your own docs site.

## Features shown

- **Sidebar navigation** with nested sections (see left)
- **Table of contents** with scroll tracking (see right on desktop)
- **Breadcrumbs** above the content
- **"View for LLM"** button to see the `.md` source
- **"Edit on GitHub"** link in the TOC sidebar (via `gitRemote` template global)
- **Dark mode** with system preference detection (toggle at bottom of sidebar)
- **Code blocks** with syntax highlighting and copy buttons
- **Version display** in the header

Browse the "Features" section in the sidebar to see each one documented.
